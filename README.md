# Aachar Ecommerce  
> **Note:** This project is for demonstration purposes only — not for commercial or public use.

A fully functional **eCommerce website** built with **Next.js**, offering a seamless shopping experience with both frontend and backend capabilities.  
The platform emphasizes **performance**, **scalability**, and **security**, featuring a modern UI and robust backend integration.

## ⚙️ Tech Stack  

**Frontend:**  
- [Next.js](https://nextjs.org/)  
- [React](https://reactjs.org/)  
- [Tailwind CSS](https://tailwindcss.com/)  

**Backend:**  
- [Next.js API Routes](https://nextjs.org/docs/api-routes/introduction)  
- [MongoDB](https://www.mongodb.com/) with [Mongoose](https://mongoosejs.com/)  
- [JWT Authentication](https://jwt.io/)  

**Other Tools:**  
- [Axios](https://axios-http.com/) for API communication  
- [Stripe](https://stripe.com/) for payments (if implemented)  
- [Cloudinary](https://cloudinary.com/) for image handling  
- [Vercel](https://vercel.com/) for deployment
